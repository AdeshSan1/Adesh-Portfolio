Bluetooth Controlled Servo and RGB LED

This project was completed most recently because I felt like I didn’t understand the fundamentals of Bluetooth controlled systems at all. In creating this Bluetooth system, I had the opportunity to create an app on my Android device that transmitted data to an HC-05 Bluetooth module, which then sent the data to an Arduino Uno. Pretty much, the plan was to use a Bluetooth module and an app to control the colour scheme of an RGB LED along with a servo motor.

You're probably wondering what's the point, right? Well, the knowledge required to develop this project can be easily applied to Android-controlled robotic arms or vehicles. These projects are on my to-do list!

The app was developed using the MIT APP Inventor platform, which was super user-friendly. In the building environment, I used code blocks to develop the app’s backend. Moreover, the front end was also designed through the platform’s UI.

To summarize the project, the LED and servo were connected to PWM pins of the Arduino Uno. By doing this, I could control the duty cycle or bits transferred by each pin. The app was the main interface that controlled how many bits would be used in each device. For example, by pressing the higher or lower buttons on the app, I was actually changing the duty cycle of the device to set the servo’s rotor angle or the RGB LED’s colour.

Project Takeaways
